# Badminton Association

A website for Badminton Association in Tsinghua University.

http://tsba.tk

![](http://luosuiqian.github.io/badminton/public/img/logo.png)

## Usage

1. `npm install`
2. `cp models/db.example.js models/db.js`
3. `vi models/db.js`
4. `./run.sh start`
